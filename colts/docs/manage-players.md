# Devon RFU Colts Administration Portal - Manage Players

## 7. MANAGE PLAYERS

**URL**: `/accounts/manage_players/`

### Overview
The Manage Players page displays all registered players for your club. This is the roster management interface for club admins.

### Page Header
- Welcome message: "WELCOME BACK, [USERNAME]"
- Club designation: "CLUB: [CLUB NAME]"

### Players Section

**Section Title**: PLAYERS

**Add Player Button**:
- **Add Player** button → Navigate to `/accounts/new_player/`
- Located at top-right of section

### Player Display Format

**Player Cards Show**:
- Player avatar/image (default green rugby player graphic)
- Player name (e.g., "BARNSTAPLE PLAYER 1")
- Player position (e.g., "PROP", "HOOKER", "LOCK")
- **Player Profile** button → Navigate to `/accounts/player_details/{id}/`


### Rugby Positions Reference

**Forward Positions**:
- PROP (1, 3)
- HOOKER (2)
- LOCK (4, 5)
- FLANKER (6, 7)
- NUMBER 8 (8)

**Back Positions**:
- SCRUM-HALF (9)
- FLY-HALF (10)
- CENTRE (12, 13)
- WING (11, 14)
- FULLBACK (15)

### Key Actions From Manage Players Page

**To Add a New Player**:
1. Click **Add Player** button at top
2. Fill in player form (see Add New Player form)
3. Enter name, age, position, bio
4. Submit

**To View Player Profile**:
1. Locate player card in grid
2. Click **Player Profile** button
3. View full player details, stats, and history

**To Edit Player Details**:
1. Click **Player Profile** button
2. On profile page, click edit option
3. Modify details
4. Save changes

**To Add Additional Players**:
1. Scroll to bottom of page
2. Click **Add Player** link
3. Fill form and submit

### Player Management Workflow

**Typical Season Start Process**:
1. Review existing player roster
2. Add new players joining the team
3. Update player positions if changed
4. Verify all player information accurate
5. Upload player photos if available

**During Season**:
- Add new players as they join
- Update player profiles with stats
- Remove players who leave (if applicable)

### Navigation

**To Access Manage Players**:
- From Home Dashboard → Club Management menu → Manage Players
- From Club Admin Dashboard → YOUR PLAYERS → VIEW PLAYERS button
- Direct URL: `/accounts/manage_players/`

**To Return to Dashboard**:
- Click **Home** in navigation bar
- Click **Club Admin Dashboard**

### Player Card Actions

**Quick Links Available**:
- View Player Profile (detailed view)
- Add Player (create new)
- View All (if on subset view)

### Footer Options

**At Bottom of Page**:
- **Add Player** link → Quick access to create new player
- Additional navigation if needed
