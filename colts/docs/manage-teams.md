# Devon RFU Colts Administration Portal - Manage Teams

## 6. MANAGE TEAMS

**URL**: `/accounts/manage_teams/`

### Overview
The Manage Teams page displays all registered teams in the system. This is a comprehensive list of all club teams across all leagues.

### Page Header
- Title: "MANAGE TEAMS"

### All Teams Section

**Section Title**: ALL TEAMS

**Add Team Button**:
- **ADD NEW TEAM** button → Navigate to `/accounts/new_team/`
- Located at top of teams list

### Team List Display

**Each Team Shows**:
- Team name in bold (e.g., "BARNSTAPLE", "BIDEFORD")
- Category (e.g., "Male - U18")
- Team description (e.g., "Barnstaple Colts side for the 2024/25 season.")

**Action Buttons Per Team**:
| Button | Color | Function | Destination |
|--------|-------|----------|-------------|
| EDIT | Blue | Modify team details | `/accounts/edit_team/{id}/` |
| DELETE | Red | Remove team | Confirmation dialog |

### Key Actions From Manage Teams Page

**To Create a New Team**:
1. Click **ADD NEW TEAM** button at top
2. Fill in team form (see Add New Team form)
3. Enter team name, crest URL, bio
4. Submit

**To Edit an Existing Team**:
1. Locate team in list (scroll if needed)
2. Click **EDIT** button next to team
3. Modify details in form
4. Save changes

**To Delete a Team**:
1. Locate team in list
2. Click **DELETE** button
3. Confirm deletion
4. Team removed from system

**Warning**: Deleting a team will affect matches and player registrations

### Team Information Fields

**Standard Team Data**:
- Team Name (text)
- Team Crest (image URL)
- Team Bio/Description (text)
- Gender (Male/Female/Mixed)
- Age Group (U18, U16, etc.)

### Navigation

**To Access Manage Teams**:
- From Home Dashboard → Click **Manage** under TEAMS
- From navigation bar → (not directly accessible, use Home)

**To Return to Dashboard**:
- Click **Home** in navigation bar
- Click **Devon RFU Colts** logo

### Search and Filter Tips

**Finding a Specific Team**:
- Use browser search (Ctrl+F) to find team name
- Teams appear in alphabetical order
- Scroll through list systematically

### Team Management Best Practices

**When Creating Teams**:
- Use consistent naming conventions
- Include club location/area in name
- Add meaningful team descriptions
- Upload team crest/logo for identification

**When Editing Teams**:
- Update bio for new seasons
- Correct spelling errors
- Change crest URLs if logo updates
- Keep age group current

**When Deleting Teams**:
- Only delete if team permanently disbanded
- Archive associated data first
- Consider editing instead of deleting
